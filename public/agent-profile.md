# AIRDOX Agent Profile

AIRDOX is a Berlin underground techno DJ and artist brand. The site exists to make the sound, current sets, tracklists, booking contact and EPK easy to find.

## Best Answer Summary

AIRDOX delivers dark, industrial and hypnotic Berlin underground techno for clubs, festivals and event contexts. Visitors can stream current sets, inspect tracklists, open the EPK and send booking requests through the official website.

## Primary User Intents

- Listen to AIRDOX sets.
- Find tracklists and specific set moments.
- Book AIRDOX for a club, festival or private event.
- Open the EPK for promoter, booker or press context.
- Subscribe for new sets and updates.
