# AIRDOX Lettering Lab Brief

## Purpose

Create a repeatable AIRDOX lettering exploration process before building more reels. The goal is custom wordmark energy, not normal font styling.

## Output

- Location: `public/brand-assets/lettering-lab/v2026-06-02/`
- Contact sheet: `contact-sheet.svg`
- Gallery: `index.html`
- Manifest: `manifest.json`

## Creative Rules

- AIRDOX letters must be built from custom shapes, paths, strokes, cuts, masks, or drawn layers.
- Do not use normal fonts for the AIRDOX wordmark. UI captions and notes may use standard type.
- Each direction must keep AIRDOX readable at reel scale.
- Best candidates should move into Photoshop/vector refinement before final Remotion animation.

## Current Directions

- Stencil Pressure
- Marker Burn
- Chrome Cuts
- Tape Wild
- Broken Neon
- Block Assembly
- Spray Skeleton
- Ripped Poster

## Next Refinement Pass

1. Pick 2-3 promising directions.
2. Redraw letter joins and counters manually, especially `R`, `D`, `O`, and `X`.
3. Add real hand texture in Photoshop: rough edges, overspray, paint drag, sticker folds, or scan artifacts.
4. Export clean transparent PNG and SVG layers.
5. Build 5s Remotion motion tests only after the letterform itself is strong.
