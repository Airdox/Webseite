# Visual Templates

Visual source material and generated design outputs live here.

- `designer/` contains lettering labs, designer portfolio output, SVG boards, contact sheets, and reusable visual directions.
- `social/` contains generated social output, preview packs, reels, captions, and upload-ready visual assets.
- `inbox/` is for uncategorized visual material that still needs review.

Runtime website assets that are served publicly stay under `public/brand-assets/`.
