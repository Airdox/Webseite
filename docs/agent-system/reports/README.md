# Reports

Human-readable planning, review, and knowledge artifacts live here.

- `operations/` is for agent operating models, architecture, governance, routing, and automation runbooks.
- `campaigns/` is for reach plans, Manni growth work, posting packs, Meta handoff notes, and social campaign briefs.
- `mentor/` is for training, learning loops, postmortems, and education advancement reports.
- `research/` is for NotebookLM outputs, research briefs, and implementation research.
- `proof/` is for visual QA screenshots and evidence gathered during reviews.

The `latest-*` files one level above are generated snapshots and stay in the workbench root unless the writer scripts are changed together.
