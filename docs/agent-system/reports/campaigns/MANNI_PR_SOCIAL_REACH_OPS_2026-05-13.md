## Ausfuehrungsrahmen

Manni darf die untenstehenden Aktionen ausfuehren oder beauftragen, sobald Plattformzugang vorhanden ist. In diesem Repository gibt es keine Instagram-/Facebook-/TikTok-/YouTube-API-Credentials und keinen Social-Publishing-Connector. Deshalb wird die Ausfuehrung lokal als freigegebener Agentenauftrag dispatcht und muss auf der jeweiligen Plattform mit den exakt freigegebenen Assets/Copies abgeschlossen werden.

Keine Abweichung ohne neue Freigabe:
- keine andere Copy
- kein anderes Asset
- keine DM-/Kommentar-Antwort ausser den freigegebenen Varianten
- kein Paid-Spend, Boost-Test oder Budgeteinsatz ohne neue explizite Freigabe
